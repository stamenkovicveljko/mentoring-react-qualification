# UserRepos Component

Creates searchable list of specified user's repos from the github.

## Implementation

It relies on `user` and `repos` to be defined.

_User model_

```ts
interface IGithubUser {
  name: string;
  login: string;
  avatar_url: string;
  html_url: string;
}
```

_Repo model_

```ts
interface IGithubRepo {
  name: string;
  html_url: string;
}
```
